C++ 核心指南目录

“Avoid the need for range checking”

理由

不会溢出的代码不溢出，（通常跑得更快）

例子 

for (auto& x : v)      // print all elements of v
    cout << x << '\n';

auto p = find(v, x);   // find x in v

强化

  • 发现显式类型检测的代码，启发式的提供改进建议
