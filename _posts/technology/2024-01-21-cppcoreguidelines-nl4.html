C++ 核心指南目录

“Maintain a consistent indentation style”

理由

可读性。避免愚蠢的错误。

坏例子 

int i;
for (i = 0; i < max; ++i); // bug waiting to happen
if (i == j)
    return i;

注意

最好在 if (...) ， for (...) 和 while (...) 之后缩进。 

if (i < 0) error("negative argument");

if (i < 0)
    error("negative argument");
