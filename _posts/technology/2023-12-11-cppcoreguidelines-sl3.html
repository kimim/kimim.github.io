C++ 核心指南目录

“Do not add non-standard entities to namespace std”

理由

添加到 std 会使得代码不符合标准，还可能导致将来的版本出错。

例子 

namespace std { // BAD: violates standard

class My_vector {
    //     . . .
};

}

namespace Foo { // GOOD: user namespace is allowed

class My_vector {
    //     . . .
};

}
