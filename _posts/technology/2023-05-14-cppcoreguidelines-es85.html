C++ 核心指南目录

“Make empty statements visible”

理由

可读性。

例子 

for (i = 0; i < max; ++i);   // BAD: the empty statement is easily overlooked
v[i] = f(v[i]);

for (auto x : v) {           // better
    // nothing
}
v[i] = f(v[i]);

强化

标记不是代码块的空语句。注释不算在内。
