C++ 核心指南目录

理由

纯函数更容易推导，更容易优化（甚至并行处理），更容易做成记忆函数（memoize）。

例子： 

template<class T>
auto square(T t) { return t * t; }

int main()
{
    cout << square(10) << endl;
    cout << square(10.1) << endl;
    return 0;
}
