C++ 核心指南目录

“Unless you use printf-family functions call ios_base::sync_with_stdio(false)”

理由

让 iostream 和 prinf 风格的 I/O 操作同步成本很高。 cin 和 cout 默认和 prinft 同步。

例子 

int main()
{
    ios_base::sync_with_stdio(false);
    // ... use iostreams ...
}
