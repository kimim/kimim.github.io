C++ 核心指南目录

“Use C++-style declarator layout”

理由

C 风格的布局强调表达式和语法，而 C++ 风格的布局则强调类型。对表达式的要求不适用于引用。

例子 

T& operator[](size_t);   // OK

T &operator[](size_t);   // just strange
T & operator[](size_t);   // undecided

注意

如果没有其他更好的想法的话，请遵循这条规则。这条规则是很多人要求添加的。

强化

  • 因为历史原因，很难强化。
