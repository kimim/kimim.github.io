C++ 核心指南目录

“Avoid names that are easily misread”

理由

可读性。并非每个人的显示器和打印机能很好区分所有字符。对于相似的拼写和细小的错拼，我们很容易误读。

例子 

int oO01lL = 6; // bad

int splunk = 7;
int splonk = 8; // bad: splunk and splonk are easily confused
