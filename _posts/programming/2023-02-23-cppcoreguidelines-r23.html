C++ 核心指南目录

“Use make_unique() to make unique_ptr’s”

理由

代码更精简。在复杂语句中，确保发生异常也是安全的。

例子 

unique_ptr<Foo> p {new Foo{7}};    // OK: but repetitive

auto q = make_unique<Foo>(7);      // Better: no repetition of Foo

强化

  • 警告：用 new 构造 unique_ptr 的地方
