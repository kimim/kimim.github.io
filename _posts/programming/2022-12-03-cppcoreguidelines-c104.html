C++ 核心指南目录

C.104: Give a container a default constructor that sets it to empty

理由

这样做可以使得容器更正规。更符合用户熟悉的习惯。

例子 

vector<Sorted_sequence<string>> vs(100);    // 100 Sorted_sequences each with the value ""
